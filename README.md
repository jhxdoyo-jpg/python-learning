# Python 学习项目

第 1-2 周 Python 完备化训练，全产出在此。

## 目录

| 文件夹 | 学习内容 | 产出 |
|--------|---------|------|
| `词频/` | 文件读写、推导式 | 词频统计脚本 |
| `调api/` | requests、JSON 解析 | 调用名言/预测年龄 API |
| `矩阵/` | NumPy、Pandas | 成绩分析、世界人口趋势图 |

## 用到的技术

- Python 3.13
- NumPy / Pandas / Matplotlib
- requests（API 调用）

## 检验

读取 JSON/CSV → 解析 → 筛选 → 统计 → 画图 → push 到 GitHub。
