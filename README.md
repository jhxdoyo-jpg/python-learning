ML入门学习笔记：
ml指的就是机器学习，不靠编程程序而是靠机器自我学习实现功能
在ml中有两种机器学习方式
一：监督学习
喂给机器实例，输入x得到y，从中学习，然后得以从新x推导出新y
就比如说判断肿瘤是否为恶性，判断是否生病
在监督学习中，有着线性回归模型(输入x，类似y=wx+b这种简单函数，从训练集中得到wb值从而估算接下来的值)
二：无监督学习
喂给机器一系列数据，让他从中找到规律或者一些有趣的东西
其中包含着聚类分组(比如说搜素新闻时通过关键词把新闻放一块，又或者是调查每一群体的意愿)异常检测(金融中找异常)降维技术(把大量数据压缩为更小的数据集)
线性回归模型中，可以通过计算得出代价函数，这个代表预测值和实际值的误差，为了使计算结果足够精准，就得使代价函数的值J(w)最低
那么如何在机器中实现得出最低的J(w)值呢，就得引出梯度下降这个算法了，为什么叫梯度下降，因为J(w)三维图中(xy轴为wb值z为J(W))中得到的图形往往像小山丘一样，得到最小的J(w)就像从高梯度往下走
如何让机器自主地往下走呢，就要用到函数关系，引入学习率和偏导数的概念
学习率:指wb值移动步子的大小，太小会导致找最小J(w)的速度过慢
太大则可能会导致找不到最小J(w)值
在计算新wb值的公式中引入了J(w)的斜率，所以在最低点左边会自动把步子往右迈，右边就往左迈，这个挺有意思的，注意一个细节梯度下降中w和b是同时更新的——用旧的w和b分别算出新的w和b然后一起替换不是先更新w再用新w去算新b。
特征缩放:不同参数的范围可能不同，有的大有的小，就会导致变化大的影响函数结果程度深，在不断迭代后范围大的赋予参数值就小，如果训练集中的数据密集在一块就会导致代价函数等高线特别密集，不方便迭代进行，这时候进行数据的缩放，可以使迭代进行更快，也就是梯度下降速度变快
特征转化也是尤其重要的一点，比如给了土地的宽度长度，这时候可以引入x3也就是x3=x1x2，因为直觉来说土地面积对价格的影响更大
多项式回归:在寻找一个函数与对应的训练数据的时候，往往不止一个项，就比如类似曲线的训练数据，这时候可以考虑很多函数如二次三次函数，还有平方根函数
往往在面对判断这类问题时，线性回归模型不能做出较好的预测，偏差比较大，这时候就要引入一个新的模型--逻辑回归模型
逻辑回归模型:1/1+e(-z) -z表示-z次方，这个模型图像为s型，能很好预测结果，最高为1，低为0，这个还能很好得出概率，是的概率是多少，注意z=wx+b这样的形式。
决策边界:顾名思义就是判断是与否的临界值，像在逻辑回归模型中临界值就得是0.5，所以z要=>0
过拟合:拟合就是指函数预测与实际值的重合程度，这个可不是越高越好，因为随着项数增多，函数与数据重合，可能会导致产出函数的波动特别大，跟我们想要的相差甚远，这时候就叫过拟合，那么如何解决过拟合呢，可以增加训练数据，人工减少一些数据不多不重要的特征，使用正则化减少参数大小
正则化=在代价函数后面加一个惩罚项惩罚太复杂的模型。其中的一个例子就是把w引入代价函数中，b虽然也可以但它影响不大
